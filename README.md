# Minecraft Classic Offline
## Notice
I do not claim to own any of the original minecraft assets used in this project
## Installation
Install python
 - For Windows, run the python installer executable from the official python download page which can be found [here](https://www.python.org/downloads/windows/)
 - For Mac, run the python installer package from the official python download page, which can be found [here](https://www.python.org/downloads/macos/)
 - For Linux, you don't need to install python because Python is installed with Linux (yay!)

Download the mcclassic.py from the latest release
Run the Python script
## Usage
To just play vanilla minecraft classic, simply press play.
To play with mods, select your mods from the list, and press play.
